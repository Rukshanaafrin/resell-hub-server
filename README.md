# ReSellHub Server

## Project Purpose

ReSellHub Server provides secure REST APIs for managing authentication, products, categories, wishlist, orders, payments, reports, and dashboard functionalities for the ReSellHub marketplace platform.

## Key Features

* JWT Authentication
* Role-Based Authorization
* MongoDB Integration
* Featured Products API
* Categories API
* Product Details API
* Wishlist API
* Orders API
* Payment API
* Stripe Integration
* Dashboard APIs
* Product Reporting System
